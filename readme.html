This article makes it easy to learn and use functions and sessions. Look at the following procedure.
Step 1
Create a database as in the following:
Create database login_db;
Step 2
Create a table as in the following:
CREATE TABLE IF NOT EXISTS `users` ( `id` int(11) NOT NULL AUTO_INCREMENT, `username` varchar(30) NOT NULL, `emailid` varchar(30) NOT NULL, `password` varchar(30) NOT NULL, PRIMARY KEY (`id`) ) ENGINE=MyISAM DEFAULT CHARSET=latin1 AUTO_INCREMENT=3 ;
